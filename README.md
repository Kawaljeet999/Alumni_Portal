# Alumni Portal

A full-stack web application designed to connect alumni, students, and administrators through a centralized platform for networking, mentorship, communication, and career opportunities.

## Features

* Secure User Authentication & Authorization
* Alumni Directory & Profile Management
* Student-Alumni Networking
* Responsive User Interface
* RESTful API Integration
* Role-Based Access Control
* Dashboard for User Management
* Scalable Client-Server Architecture

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

## Installation

### Clone Repository

```bash
git clone https://github.com/Kawaljeet999/Alumni_Portal.git
cd Alumni_Portal
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```
